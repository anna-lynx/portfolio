Welcome to my course project portal page!

These projects were created as part of my journey through front-end web design and development.

Main course projects

  • My First Page

    This was my very first HTML page. I practiced creating basic structure — adding headings, paragraphs, images, and links. It was my intro to writing real HTML code and understanding how websites are built from scratch. I also explored how small visual choices like font styles, spacing, and layout can affect how a page feels. While the result was simple, it marked the beginning of my creative and technical journey into web design.

  • Squint Studio Build-Along

    This project was a step-by-step build-along where we created a fictional design studio website. I learned how to structure more complex layouts using semantic HTML and began incorporating CSS for design and layout. This helped me get more confident working with full-page designs.

  • Smart Care

    This project was created before I started my web design course, and it marked my first full attempt at building a complete website from scratch. I explored layout, navigation, and visual structure on my own, experimenting with what I knew at the time. It’s a raw but meaningful milestone that shows where my journey began.

Helpful Resources for Web Design and Development

  • W3Schools

    A beginner-friendly platform that explains HTML, CSS, JavaScript, and more with interactive examples and live editors. Great for quick reference and practice.

  • Mimo: Python, JavaScript, HTML, CSS, and more

    A mobile-first learning app that teaches programming through bite-sized lessons and real-world projects. Perfect for learning on the go and staying consistent.

  • Sololearn: AI & Code Learning

    An interactive coding platform with lessons, quizzes, and challenges in multiple languages. Also includes community Q & A and AI-powered explanations.
